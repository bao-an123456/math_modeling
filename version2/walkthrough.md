# 绿色物流车辆调度优化 — 结果报告

## 数学模型

### 模型类型
**带软时间窗的异构车队车辆路径问题 (HFVRPTW)**

### 目标函数

$$\min Z = C_{启动} + C_{能耗} + C_{碳排} + C_{等待} + C_{迟到}$$

| 成本项 | 公式 |
|--------|------|
| 启动成本 | $400 \times$ 使用车辆数 |
| 能耗成本 | $\sum$ 各段 FPK/EPK × 载重系数 × 距离 × 单价 |
| 碳排放成本 | $0.65 \times$ 总碳排放量 |
| 等待成本 | $20 \times \sum \max(0, a_i - s_i)$ |
| 迟到惩罚 | $50 \times \sum \max(0, s_i - b_i)$ |

### 约束条件
1. 每个客户(含拆分)恰好被服务一次
2. 载重约束: 路径总重量 ≤ 车辆载重上限
3. 体积约束: 路径总体积 ≤ 车辆容积上限
4. 车辆数量约束: 每类使用量 ≤ 可用量
5. 流守恒: 从仓库出发, 返回仓库

### 时变速度模型
分段处理跨时段旅行，各时段使用均值速度

### 能耗模型
- U型曲线 + 按载重比例线性插值负载因子

---

## 求解算法: ALNS

1. **初始解**: 按时间窗排序的贪心插入法
2. **破坏算子**: 随机移除、最差移除、Shaw相似度移除
3. **修复算子**: 贪心插入、遗憾值插入
4. **接受准则**: 模拟退火
5. **自适应权重**: 每100次迭代更新算子选择权重
